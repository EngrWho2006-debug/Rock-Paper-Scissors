# Rock-Paper-Scissors
A fun and interactive Rock Paper Scissors game built using HTML, CSS and JavaScript. Players can compete against the computer by choosing Rock, Paper or Scissors.

## Features
- Rock, Paper, Scissors gameplay.
- Computer-generated random moves.
- Win, Loss, and Draw tracking.
- Round counter.
- Reset Score Button.
- Popup messages for game results.
- Responsive design for desktop and mobile devices.

## Technologies used
. HTML5
. CSS3
. JavaScript(ES6)

## How to Play
1. Select Rock, Paper, or Scissors.
2. The computer will make a random choice.
3. The winner is decided according to the game rules:
       > Rock beats Scissors
       > Scissors beats Paper
       > Paper beats Rock
4. Scores and rounds update automatically.
5. Click the Reset button to start over.
